# 🤖 VoxMind AI

🚀 Voice-First Multilingual Intelligent Chatbot powered by AI + Murf Voice

---

## 🌟 Overview

VoxMind AI is an advanced AI chatbot that enables natural human-like interaction through both text and voice.  
Unlike traditional bots, it does not rely on predefined responses but generates intelligent answers dynamically.

---

## 💡 Key Features

- 🎤 Voice Input (Speech-to-Text)
- 🔊 Voice Output using Murf AI (Human-like speech)
- 🌍 Multilingual Support (English, Hindi, Telugu)
- 🤖 AI-powered responses (real-time)
- 💬 Interactive chat UI with animations
- 🎧 Sound wave animation during voice playback

---

## 🧠 How It Works

1. User gives input (text/voice)
2. OpenAI generates response
3. Response sent to Murf AI
4. Murf converts text → speech
5. Audio + text displayed to user

---

## 🛠️ Tech Stack

- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- AI Engine: OpenAI API
- Voice Engine: Murf AI

---

## 🚀 Unique Factor

This project focuses on **voice-first interaction**, combining AI intelligence with studio-quality speech synthesis to enhance user experience.
